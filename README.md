# Lars Vault

Markdown-first operating system for AI-assisted work, business projects, prompts, workflows, and strategic context.

## Purpose

This repository is designed to be readable by both humans and AI tools.

It will store:

- operating principles
- active projects
- reusable prompts
- AI workflows
- venture-building playbooks
- technical setup notes
- decision logs

## Structure

```text
principles/
projects/
ventures/
people/
playbooks/
prompts/
agents/
systems/
research/
meetings/
finance/
health/
europe/
ideas/
archive/
```

## AI Agent Rules

1. Read the context files before executing tasks.
2. Prefer clear Markdown.
3. Preserve strategic context.
4. Do not overwrite important decisions without explicit instruction.
5. Keep files simple, portable, and easy to search.
