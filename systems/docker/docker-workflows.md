# Docker Workflows

## Objective

Use Docker containers to safely run autonomous AI agents and experiments.

## Recommended Use Cases

- autonomous coding
- dependency upgrades
- risky refactors
- rapid experimentation
- isolated testing

## Suggested Container Stack

```text
Container
 ├── Node.js
 ├── Python
 ├── Git
 ├── Supabase CLI
 ├── Vercel CLI
 ├── Claude Code
 └── Codex
```

## Benefits

- isolation
- reproducibility
- rollback safety
- reduced operational risk
- cleaner experiments

## Operational Principle

Dangerous autonomous execution should happen inside isolated environments whenever possible.
