# SaaS Launch Playbook

## Objective

Reduce the time between idea and market validation.

## Workflow

### Phase 1 — Positioning

- define ICP
- define pain point
- define value proposition
- define CTA

### Phase 2 — Landing Page

- create simple page
- clear messaging
- mobile-first
- lead capture

### Phase 3 — Validation

- generate traffic
- outreach
- collect leads
- schedule conversations
- evaluate traction

### Phase 4 — MVP

- build minimal product
- prioritize onboarding
- avoid unnecessary features
- deploy quickly

### Phase 5 — Iteration

- customer feedback
- UX improvements
- automation
- analytics

## Core Principle

Launch before perfect.
