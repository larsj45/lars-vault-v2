# Prompt Library

Reusable prompts for development, business, strategy, content, and autonomous workflows.

## Categories

### Development
- landing pages
- SaaS architecture
- Supabase setup
- debugging
- autonomous coding

### Business
- positioning
- outreach
- partnerships
- proposals
- validation

### Content
- LinkedIn posts
- websites
- presentations
- newsletters

### Agent Prompts
- autonomous agents
- coding agents
- workflow agents
- research agents

## Prompt Design Principles

Good prompts should:
- provide context
- define objectives
- define constraints
- encourage autonomy
- minimize ambiguity
- optimize for execution

## Standard Structure

```text
Context
Objective
Constraints
Expected Output
Operational Rules
```
