# Operational Philosophy

## Core Principle

Execution matters more than ideas.

The objective is to create systems that:
- accelerate execution
- reduce friction
- maximize leverage
- increase optionality
- compound knowledge over time

## Strategic Principles

### Speed Over Perfection
- launch early
- iterate aggressively
- avoid overengineering
- optimize for momentum

### MVP-First Thinking
Every new venture should start with:
1. positioning
2. landing page
3. lead capture
4. validation
5. MVP

### Parallel Experimentation
Run multiple asymmetric bets simultaneously.

### AI-Native Operations
AI is operational infrastructure, not merely an assistant.

### International Optionality
Build systems with geographic flexibility and portability.

## Working Style

Favor:
- practical decisions
- reversible decisions
- fast iteration
- simplicity

Avoid:
- perfection paralysis
- unnecessary complexity
- excessive planning
